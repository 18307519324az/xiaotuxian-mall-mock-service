# 数据说明

Mock Service 使用本地 JSON 数据为前端提供演示接口数据。

## 数据目录

```text
src/main/resources/mock/
data/
```

`src/main/resources/mock/` 保存固定响应与基础数据，`data/` 保存服务运行时产生的演示状态。

## 数据范围

数据覆盖以下业务场景：

- 首页
- 分类
- 商品
- 用户
- 购物车
- 地址
- 订单
- 收藏与浏览历史
- 评价与售后
- 优惠券、礼品卡与积分

## 数据特点

- 不依赖外部数据库
- 适合本地启动和前端联调
- 便于演示商城常见业务流程
- 支持通过接口重置本地演示状态

## 运行时数据

服务运行过程中会在 `data/` 下生成运行时数据文件。运行时文件由服务维护，不作为固定基础数据使用。

## 重置数据

```bash
curl -X POST http://localhost:8099/admin/reset-member-data
```
